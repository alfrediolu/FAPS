BIOSUN Lab Protein Database

This database contains secondary structure and sequence length data for proteins.
The data source is marked under SIM/Uniprot, as the database contains both our simulated values and experimental values taken from Uniprot.


Single-Protein Lookup

Enter a single accession code to filter results. Leaving blank will bring up all entries in database.

CSV-Based Search

Upload a .csv file with a column name 'Accession' to use as a list to search with.
